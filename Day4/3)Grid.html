Explanation of CSS Grid Properties

  • display: grid; - Defines the element as a grid container.
  • grid-template-columns: - Specifies the number and size of columns in the grid.
  • grid-template-rows: - Specifies the number and size of rows in the grid.
  • gap: - Defines the space between grid items.
  • grid-column: - Specifies the starting and ending column lines for a grid item.
  • grid-row: - Specifies the starting and ending row lines for a grid item.
  • grid-column: span [number]; - Makes the grid item span across the specified number of columns.
  • grid-row: span [number]; - Makes the grid item span across the specified number of rows.
