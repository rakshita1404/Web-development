CSS Position Properties

This page demonstrates the different CSS position properties with examples and comments.

Static Position

The position: static; property is the default position for HTML elements. It is not affected by the top, bottom, left, or right properties.

This is a static positioned box.

Sticky Position

The position: sticky; property toggles between relative and fixed, depending on the scroll position. It is positioned relative until a given offset position is met in the viewport - then it "sticks" in place (like position:fixed).

This is a sticky positioned box.

Relative Position

The position: relative; property positions the element relative to its normal position. The top, right, bottom, and left properties will cause it to be adjusted from its normal position.

This is a relatively positioned box.

Absolute Position

The position: absolute; property positions the element relative to its nearest positioned ancestor (not static). If there is no such ancestor, it uses the document body, and moves along with page scrolling.

This is an absolutely positioned box.

Fixed Position

The position: fixed; property positions the element relative to the browser window. It will not move when the page is scrolled.

This is a fixed positioned box.
