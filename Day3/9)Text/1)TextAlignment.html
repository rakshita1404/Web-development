CSS Text Alignment Properties

Center Alignment

This text is centered using the "text-align: center;" property.

Left Alignment

This text is left-aligned using the "text-align: left;" property.

Right Alignment

This text is right-aligned using the "text-align: right;" property.

Justify Alignment

This text is justified using the "text-align: justify;" property. Justified text is spaced so that the left and right sides of the text block both have a straight edge.
